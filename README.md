# Consumidor de Produtos

Este projeto consiste em uma aplicação web para consumir uma API REST de produtos e exibir uma lista de produtos da loja. A aplicação é composta por uma única página/rota contendo uma seção da loja e um carrinho de compras.

## Funcionalidades

A aplicação possui as seguintes funcionalidades:

- **Loja**: A lista de produtos é buscada de uma API REST, enquanto estiver carregando, é exibido um efeito de carregamento.
  
- **Carrinho**: Permite adicionar produtos selecionados ao carrinho e aumentar a quantidade de cada produto.

### Tecnologias utilizadas:

- TypeScript
- React (Next.js)
- Styled-components

## Como rodar a aplicação

Para rodar a aplicação, siga os passos abaixo:

1. Clone o repositório do projeto:

```
git clone git@github.com:MayaraRocha95/MKS-Frontend-challeng.git
```

2. Acesse o diretório do projeto:

```
cd MKS-Frontend-challeng
```

3. Instale as dependências:

```
npm install
```

4. Inicie a aplicação:

```
npm start
```

5. Acesse a aplicação no navegador através do link: [http://localhost:3000](http://localhost:3000)
